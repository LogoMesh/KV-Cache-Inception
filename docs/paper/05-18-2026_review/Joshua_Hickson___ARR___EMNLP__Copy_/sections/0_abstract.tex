Inference-time tree search over a frozen Transformer's KV cache offers a gradient-free route to test-time scaling, yet its behavior under surrogate-reward guidance remains poorly understood. We introduce \textbf{Reversible KV-Cache MCTS}, an inference-time algorithm that mutates the KV cache in place using FP32 accumulators and exactly rolls back after each tree expansion. We prove zero-drift reversibility, validated to machine precision over 200 apply-revert cycles on Llama 3.2-1B/3B, and show an $O(M_{KV}+d\cdot K_{acc})$ working-set memory complexity independent of branching factor, in contrast to the $O(b^d\cdot M_{KV})$ cost of naive parallel-cache MCTS.

Using this infrastructure, we reveal a \textbf{scale-conditional failure mode} of surrogate-reward latent-space search, which we frame as reward-driven Goodharting in KV-cache geometry, or \emph{Dimensional Escape}. On ARC-Easy, an entropy-minimization reward that is statically correlated with correctness fails to improve over a uniform-random reward at both 1B and 3B scales, but for different reasons: the 1B model amplifies a first-letter prior, while the 3B model collapses toward greedy decoding. A latent cartography sweep across prompt classes further shows that this mechanism depends on the model's prior structure. Reversible KV-Cache MCTS thus provides an interpretability tool for studying continuous latent search in frozen Transformers, complementing monitor-driven gradient-input analyses.
